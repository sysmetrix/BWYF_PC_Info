사내 PC 네트워크 정보 제출 안내

업무 생산성 향상을 위한 AX 챌린저 자동화 도구 배포

안녕하십니까, 임직원 여러분.

재단의 효율적인 IT 보안 관리와 신속한 네트워크 유지보수 지원을 위해, 클릭 한 번으로 "내PC 지킴이"가 설치된 컴퓨터의 정보를 확인할 수 있는 '자동 수집 도구'를 배포합니다.

안내에 따라 파일을 다운로드하여 실행해 주시면 즉시 정보가 복사됩니다. 원활한 관리를 위해 기한 내에 꼭 제출해 주시기를 부탁드립니다.

🚨 실행 시 '보안 경고' 창이 나타난다면?

본 도구는 사내에서 자체 제작한 안전한 파일이지만, 상용 프로그램이 아니기 때문에 윈도우 기본 방어막인 "게시자를 확인하지 못했습니다" 또는 "PC 보호" 경고가 나타날 수 있습니다.

정상적인 현상이며 PC에 무리를 주지 않으니, 안심하시고 [추가 정보] → [실행] 또는 [확인] 및 [실행(R)] 버튼을 눌러 진행해 주시면 됩니다.

💡 도구 사용 및 제출 방법

  1. 아래의 [⬇️ 확인 도구 다운로드] 버튼을 클릭하여 압축 파일(.zip)을 다운로드합니다.
  2. 다운로드한 파일의 압축을 푼 뒤, 폴더 안에 있는 실행 파일(.bat)을 더블 클릭합니다.
  3. 검은색 창이 열리면 PC 사용자 본인의 이름을 입력하고 엔터(Enter)를 누릅니다.
  4. 화면에 진단 결과가 나타남과 동시에, 제출할 정보가 눈에 보이지 않게 자동으로 복사됩니다.
  5. 실행창을 닫고, 아래 [📝 PC 정보 제출하기] 버튼을 눌러 폼에 접속한 후, 입력창에 붙여넣기(Ctrl+V) 하여 제출하면 완료됩니다!
